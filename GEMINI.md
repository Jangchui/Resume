# Gemini Workspace

This directory is ready for your project.

## Getting Started

- **Add your files:** Place your project files in this directory.
- **`GEMINI.md`:** This file provides context to the Gemini CLI. You can edit it to describe your project.

## Example `GEMINI.md` for a Non-Code Project

If this were a directory for your resume, the `GEMINI.md` file might look like this:

```markdown
# Resume

This directory contains the source files for my resume.

## Key Files

- `resume.tex`: The LaTeX source file for my resume.
- `profile.jpg`: My professional headshot.
- `references.bib`: A BibTeX file with my professional references.

## Usage

To build the resume, use a LaTeX compiler:

```bash
pdflatex resume.tex
```
```
